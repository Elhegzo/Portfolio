# Mostafa Hegazy — Portfolio

Personal portfolio website for Mostafa Hegazy, Product Manager based in Montreal, QC.

**Live site:** https://elhegzo.github.io/Portfolio/

## About

Single-page portfolio built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools. Designed with a dark editorial aesthetic featuring a custom cursor, scroll-reveal animations, and an interactive product stories modal.

## Sections

- **Hero** — Name, tagline, key stats, and CTAs
- **About** — Bio, profile photo, and quick-info sidebar
- **Work History** — Experience timeline with role bullets and impact pills
- **By The Numbers** — Animated impact metrics
- **How I Think** — Clickable product stories in STAR format
- **Skills** — Product and tooling skill groups
- **Contact** — Email, LinkedIn, and resume download

## Files

| File | Description |
|------|-------------|
| `index.html` | Entire site — HTML, CSS, and JS in one file |
| `Mostafa Hegazy PM.pdf` | Resume (linked from Download Resume buttons) |
| `profile.jpg` | Profile photo (add to repo root to display in About section) |

## Adding Your Profile Photo

Upload your photo to the repo root as `profile.jpg`. It will automatically appear in the About sidebar with a portrait crop and subtle desaturate treatment.

## Tech

- Vanilla HTML/CSS/JS
- Google Fonts: Bebas Neue, DM Sans, DM Mono
- No dependencies, no build step
- Hosted via GitHub Pages
